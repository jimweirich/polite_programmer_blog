1. Customizing this template
-----------------------------------------
To change the text, content, links etc. open the index.html file in your preferred HTML editor.
It can be notepad, dreamweaver, frontpage or any of your favorite HTML editor.

To change the CSS open styles.css in a HTML/CSS editor and change the appropriate 
values to suit your needs.

All images are stored in the "images" folder.
